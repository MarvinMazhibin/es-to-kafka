# es_to_kafka

```
一个现有的项目如何传到github，步骤如下：
在github网站操作：在github上新建一个仓库

在mac本地操作：echo "# es_to_kafka" >> README.md # 先添加一个MD文件 用于描述项目
vi .gitignore # 创建.gitignore文件，并写入内容
git init # 把这个目录变成一个git仓库
git add . # add这个路径下所有文件（追踪除了ignore的文件）
git commit -m '哈哈 创建第一次commit' 
git remote add origin git@github.com:MarvinMazhibin/es_to_kafka.git
git push -u origin master
```
